# 井字號，後面為解釋

- `git status`  # 檢查狀態
- `git add (檔名)`  # 將未被追蹤的檔案，索引區內的檔案添加到暫存區
- `git commit -m "上傳的訊息"`  # 記得前後要加雙引號
- `git push`  # 上傳到 GitHub，現在預設分支為 master
- `git push origin master/main`  # 當然你也可以推到 main 分支，但是要指定
